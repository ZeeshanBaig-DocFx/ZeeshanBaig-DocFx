Default views and Custom views

The default choices to view files and folders in the contents pane of File Explorer are viewing it by different sizes, details, lists, tiles, and contents. SharePoint is not restricted to default views but have customs views which can be created according to requirements.

You can create custom views of a document library in SharePoint to display items in a way that makes sense for you. For example, you can select the columns you want to display and arrange them in the order you want. You can also sort, filter, or group items to create a view that suits your needs. When you're done, just give the view a name and save it. The default view, created by SharePoint, is called "All Items" in lists and "All Documents" in libraries. It is a basic view, but you can create new views and use filters, styles and grouping to arrange data in ways that suit your organization.


Group Items

Grouping gathers like items or documents based on a column value. When you group a view, it only rearranges how the data is displayed; it doesn't add or remove content from the view.

It helps you to gather all items based on a single column value. For example we can group items in a list based on cost, service, model name, etc.

Sorting and Filtering items

Sorting and Filtering of items can be done based on the column in a SharePoint document library.

Columns can be sorted either A to Z or vice versa.
