Folders vs Document Library

Folders in file share represent a physical separation for different types of documents. They can be based on many criteria depending on the organizational structure. These physical separations in the world of SharePoint are called Document Libraries. Below, we see how a folder structure in file share can be converted into SharePoint document libraries

SharePoint Document Library features

SharePoint bring many improvements from a File share Folder structure. Some of those features are highlighted below:

Granular Permission

Permissions in SharePoint gives a high degree of access control than a folder.

SharePoint File Share
The permissions in SharePoint can be broken down into site, library or even folder level. We have standard default groups called as Members, Visitors, Owners to which users are added.We can also create custom SharePoint groups to include ad hoc collection of people you want to be able to provide access in certain sites. The standard permission types available for all files and folders in File Share are Modify, Read & Execute, Write, Full Control.
Document templates

Any pre-formatted document that is created with customized headers, footers, fonts are called as document templates. These templates help in maintaining uniformity and also saves time since you need not spend time formatting a document each time you use it.

Bulk editing feature

The bulk edit feature allows the user to select multiple documents at the same time and edit the file properties from the Information Panel.

Export to Excel

In File Share, we have Excel files that act as spreadsheets with values, tables, etc. In SharePoint we call that as lists which has a collection of data. The export to Excel feature in SharePoint helps the user to export the list content to a Excel Workbook where the user can further use the content to create any charts, pivot tables in Excel. There is a one-way connection between the SharePoint and the Excel. So, any changes made in SharePoint list will be reflected in Excel workbook and not vice versa.

Alerts

The alert feature in SharePoint allows the user to get notifications through email or text message to any content changes made in the list or library. By content changes we mean any document uploaded, downloaded, deleted, or modified.
