Content types

In file share, contents inside a folder share the same level of details depending on the property columns that are enabled as follows:

SharePoint allows us to add different types of documents within the same Document Library. Each type of document requires different levels and types of details. Some can be required details and other can be optional. This allows for several enhancements such as Group by/Sort by content types and Views by content types, allow for all types of documents required in a project, each having their own details.
